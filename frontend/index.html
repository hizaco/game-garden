🌌 Comment jouer à Quantum Garden

🎯 Objectif

Cultivez un jardin quantique en observant les cellules pour faire s'effondrer leur état quantique.

🔬 Mécaniques Quantiques

  • Superposition : Les cellules existent dans plusieurs états simultanément
  • Observation : Cliquer sur une cellule fait s'effondrer son état
  • Intrication : Les cellules voisines sont affectées par l'observation

🎮 Contrôles

  • Cliquez sur une cellule pour l'observer
  • Atteignez les objectifs pour gagner
  • Coopérez avec l'autre joueur
