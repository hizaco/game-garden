🎯 Comment jouer à Battle Dots

🎮 Règles du jeu :

  • Objectif : Contrôle 60% du territoire ou élimine tous les dots adverses
  • Placement : Place un dot par tour sur une case vide
  • Expansion : Après 3 tours, tes dots s'étendent automatiquement
  • Capture : Encercle les dots adverses pour les capturer
  • Timer : 15 secondes maximum par tour

⚡ Cases spéciales :

  • 🔥 Double Placement : Place 2 dots au lieu d'1
  • 🛡️ Bouclier : Protection contre la capture
  • 💥 Expansion : Expansion immédiate

🏆 Stratégies :

  • Contrôle le centre pour une meilleure expansion
  • Crée des formations défensives
  • Utilise les coins et bords à ton avantage
  • Anticipe les mouvements de ton adversaire
